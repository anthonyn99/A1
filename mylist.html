Install PriceWatch

PriceWatch is the browser extension that reads real prices off each store's own search page, using this browser's normal session — no server, no paid data API. Browsers won't let a website install an extension silently, so this packages the latest source and walks you through the one-time load.

  1. Download the extension
    Always packages the newest files straight from GitHub.
  2. Unzip it to a folder you'll keep (e.g. Documents). The extension runs from that folder — don't delete it afterwards.
  3. Open your browser's extensions page (Chrome — a click can't open it, so copy & paste):
    chrome://extensions
    ⚠ Firefox doesn't support this extension's MV3 background worker or externally_connectable. Use Chrome, Brave, Edge or Opera.
  4. Turn on Developer mode (toggle, top-right of that page).
  5. Click Load unpacked and select the unzipped PriceWatch folder.
  6. Come back here and reload — the strip under your stores should turn into PriceWatch active. Tracking or checking an item now opens each store in a background tab, reads the price, and closes it.
  7. Already installed? Unpacked extensions don't self-update. Download again, replace the old PriceWatch folder in place, then hit Reload ⟳ on the extensions page.

    Note: stores are checked at most once every few hours each, one tab at a time, so the traffic looks like ordinary browsing. A store that shows a bot check is skipped and backed off — never retried in a loop.
